## 用户使用说明

### 教师发布任务流程
1. 进入任务发布页面 (`/teacher/tasks/create`)
2. 选择任务周期类型：
   - **单日**: 选择具体日期
   - **周任务**: 选择周，系统自动记录周一日期
   - **月任务**: 选择月，系统自动记录1号日期
3. 填写其他任务信息
4. 发布任务

### 周期类型说明
- **单日任务**: 适用于短期、具体的日期任务
- **周任务**: 适用于每周重复或持续一周的任务
- **月任务**: 适用于月度计划或长期任务
